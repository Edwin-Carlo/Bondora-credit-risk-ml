# Bondora Credit Risk ML

Proyecto de clasificación de riesgo crediticio usando datos de Bondora P2P lending.

## Estructura del proyecto

- `notebooks/`: Cuadernos Jupyter de análisis y modelado
- `data/raw/`: Datos originales sin modificar
- `data/processed/`: Datos limpios y transformados
- `artifacts/`: Modelos guardados y resultados
